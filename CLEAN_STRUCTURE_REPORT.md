# Clean Structure Report — OverCR v0.0.3-alpha

## Purpose

This package is intended to be a reusable clean-core release, not a live workspace dump.

## Included

- doctrine files
- boot prompt templates
- HQ route scaffolding
- clean config templates
- install/runtime documentation
- empty structural directories for memory, tasks, workspace, and TUI assets

## Excluded

- saved sessions
- logs
- checkpoints
- cache files
- local databases
- lead data or deployer-specific project data
- API keys, tokens, `.env` files, and credentials
- local release archives

## Authority Model Preserved

1. Filesystem state
2. Config and route memory files
3. Hermes memory
4. Chat history

## Governance Preserved

OverCR may research, read, summarize, enrich, score, draft, and organize. It may not send, submit, call, message, book, purchase, or bypass access controls without explicit operator approval.

## Public Release Notes

This release is Hermes-first. Open WebUI is described as optional and secondary. Other routers and harnesses may be used at the operator's discretion.
