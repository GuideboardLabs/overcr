## Operator Workflow

1. Extract release archive
2. Read INSTALL.md for setup
3. Run scripts/check_v2_integrity.py to verify system
4. Run tests/run_all.py to verify all tests pass
5. Explore examples/ for demo scripts
6. Read references/ for governance and architecture

## TUI Usability

- Observatory mode (not cockpit — no auto-approve)
- ApprovalAction is data display, not execution trigger
- All views support render_plain() fallback
- Rich rendering requires Unicode-capable terminal
